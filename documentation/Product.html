Methods

addToCart(targetCart, quantity) → {Product}

Add the product, in the specified quantity, to a target shopping cart. The Product class has a dedicated 'add to cart' method since, logically, the action starts with the product.

Internally, this method calls the addProduct method of the ShoppingCart class. This way, except for one interface method, this method does not need to care about how the shopping cart class is implemented.
Parameters:
Name Type Default Description
targetCart ShoppingCart The shopping cart to which the product is added.
quantity number 1 Optional. The number of products to add.
